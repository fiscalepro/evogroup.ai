# Builder — Раунд 1: BDD Сценарии

## Область 1: Валидность JSON файлов

### Сценарий: Все KGZ файлы содержат валидный JSON [SOLID]
ДАНО: 17 JSON файлов в public/locales/kgz/
КОГДА: парсер читает каждый файл
ТОГДА: все файлы парсятся без ошибок

НЕ ДОЛЖНО:
- Ни один файл не должен быть пустым (кроме analyricsshowcase.json и portfoliogallery.json — пусты и в ru)
- Не должно быть синтаксических ошибок JSON (незакрытые скобки, лишние запятые)

ГРАНИЧНЫЕ ЗНАЧЕНИЯ:
- Файл с 1 символом (пустой объект `{}`)
- Файл с вложенными объектами глубиной 5+

---

## Область 2: Полнота перевода

### Сценарий: Каждый KGZ файл содержит текст на кыргызском языке [SOLID]
ДАНО: KGZ файл с текстовыми значениями
КОГДА: проверяется язык текста
ТОГДА: все текстовые значения на кыргызском (не русский, не английский)

НЕ ДОЛЖНО:
- Ни один файл не должен содержать полностью русский текст (как было с calculator.json)
- Текстовые значения не должны быть пустыми строками

### Сценарий: Структура KGZ файлов совпадает с RU [SOLID]
ДАНО: пара файлов ru/{name}.json и kgz/{name}.json
КОГДА: сравниваются ключи JSON
ТОГДА: набор ключей идентичен

НЕ ДОЛЖНО:
- Не должно быть ключей в ru, отсутствующих в kgz (пропущенный перевод)
- Не должно быть лишних ключей в kgz, отсутствующих в ru

## Область 3: Брендинг

### Сценарий: Название компании корректно во всех файлах [SOLID]
ДАНО: все KGZ JSON файлы
КОГДА: ищется текст "ФискалеПро" или "fiscalepro"
ТОГДА: результат — 0 совпадений

НЕ ДОЛЖНО:
- "ФискалеПро" не должно появляться нигде в kgz файлах
- "fiscalepro" не должно появляться в email, telegram, URL

### Сценарий: Контактные данные соответствуют Evolution Group [SOLID]
ДАНО: файлы footer.json, contact.json
КОГДА: проверяются email, telegram
ТОГДА: email = info@evogroup.ai, telegram = @evogroup_support

## Область 4: Грамматика кыргызского языка

### Сценарий: Притяжательные суффиксы после "и" [SOLID]
ДАНО: текст содержащий аббревиатуры на "и" (ИИ)
КОГДА: добавляется притяжательный суффикс
ТОГДА: используется "-нин" (не "-нын")

НЕ ДОЛЖНО:
- "ИИнын" — неверная форма
- "Компанияның" — казахская форма (должно быть "Компаниянын")

### Сценарий: Прилагательные от заимствований [SOLID]
ДАНО: заимствованное слово (проактив, корпоратив)
КОГДА: образуется прилагательное
ТОГДА: суффикс "-дик" (не "-дүү")

НЕ ДОЛЖНО:
- "корпоративдүү" — неверно (правильно: корпоративдик)
- "проактивдүү" — неверно (правильно: проактивдик)

### Сценарий: Локатив после "и" [SOLID]
ДАНО: слово заканчивающееся на "и" (ИИ)
КОГДА: добавляется местный падеж
ТОГДА: суффикс "-де" (не "-дә")

НЕ ДОЛЖНО:
- "ИИдә" — казахская форма (правильно: ИИде)

## Область 5: Терминологическая согласованность

### Сценарий: Единая терминология по всем файлам [SOLID]
ДАНО: термин встречающийся в нескольких файлах
КОГДА: проверяется написание
ТОГДА: написание одинаковое везде

НЕ ДОЛЖНО:
- "Жумуштоо убактысы" и "Иштөө убактысы" одновременно — только второе
- "адисташкан" и "адистешкен" одновременно — только второе
- "корпоративдүү" и "корпоративдик" — только второе

## Область 6: Должности и роли

### Сценарий: CTO обозначен корректно [SOLID]
ДАНО: team.json, раздел cto
КОГДА: проверяется поле position
ТОГДА: содержит "Техникалык директор" (не "Башкы директор")

НЕ ДОЛЖНО:

## Область 7: Чат-бот API

### Сценарий: test.evogroup.ai проходит CSRF проверку [SOLID]
ДАНО: запрос с Origin: https://test.evogroup.ai
КОГДА: API проверяет isAllowedOrigin
ТОГДА: возвращает true, запрос проходит

НЕ ДОЛЖНО:
- test.evogroup.ai не должен получать 403 "Запрос отклонён"

### Сценарий: Бот отвечает на языке пользователя [SOLID]
ДАНО: фронтенд передаёт locale="ky"
КОГДА: API формирует запрос к OpenClaw
ТОГДА: сообщение содержит инструкцию [Respond in Kyrgyz]

НЕ ДОЛЖНО:
- Бот не должен отвечать на русском когда UI на кыргызском
- Бот не должен отвечать на русском когда UI на английском

### Сценарий: Ошибки отображаются на языке пользователя [SOLID]
ДАНО: API вернул ошибку, locale="ky"
КОГДА: фронтенд показывает сообщение
ТОГДА: текст на кыргызском языке

НЕ ДОЛЖНО:
- Хардкод-русские ошибки не должны показываться пользователю с другим языком
